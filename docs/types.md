# Trajectory types

We standardize on four trajectory types:

- `space`
- `air`
- `ground`
- `sea`

If no type is provided, it is treated as `other` in the UI.

# Velocity profile types

- `subsonic`
- `supersonic`
- `hypersonic`
- `suborbital`

# Maneuverability types

- `waypoint`
- `no-waypoint`
- `other`

# Convergence types

- `nominal`
- `lofted`
- `depressed`
- `other`
